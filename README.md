## 👤 Autoria e Histórico
Este sistema foi desenvolvido por **Adiel Santos Fontes**, profissional com ampla experiência em engenharia de software e arquitetura de sistemas.

*   **Autor:** Adiel Santos Fontes
*   **Data Inicial do Projeto:** Setembro de 2011

O **Retaguarda-Frente-de-Caixa** é uma solução completa para gestão de pontos de venda (PDV) e administração centralizada (Backoffice), projetada para oferecer performance e confiabilidade no controle comercial.

## 🚀 Funcionalidades do Sistema
- **Frente de Caixa (PDV):** Interface otimizada para agilidade no atendimento ao cliente e fechamento de vendas.
- **Gestão de Retaguarda:** Controle administrativo completo, incluindo cadastro de produtos e gestão de estoque.
- **Módulo Financeiro:** Acompanhamento de entradas, saídas e conciliação de caixa.
- **Consultas e Relatórios:** Ferramentas para análise de vendas e monitoramento de desempenho em tempo real.
- **Segurança e Acesso:** Sistema de autenticação integrado para diferentes níveis de usuários.

## 🛠️ Stack Técnica
- **Linguagem Principal:** C# (ASP.NET)
- **Interface:** Web Forms com suporte a Master Pages para consistência visual.
- **Configuração:** Estrutura baseada em `Web.config` para ambientes escaláveis.
- **Arquitetura:** Separação lógica entre módulos operacionais (Frente) e administrativos (Retaguarda).

## 📂 Estrutura de Pastas
*   `/Frente_de_Caixa`: Telas e lógica voltadas ao ponto de venda.
*   `/Retaguarda`: Módulos de administração e configuração do sistema.
*   `/Consultas`: Motores de busca e filtros de dados.
*   `/Relatorios`: Geração de documentos gerenciais.
*   `/App_Code`: Classes de negócio e integração de dados.
